Requirements for building the library.

  • Microsoft Visual Studio
  • Linux or Cygwin

Requirements for building help.

  • Install Doxygen for building the API docs from http://www.doxygen.org.
  • For compiling html files into a single chm file, install a chm compiler, such as hhc.exe that comes with the HTML Help Workshop for building HTML help 1.x files from http://msdn2.microsoft.com/en-us/library/ms669985.aspx.
